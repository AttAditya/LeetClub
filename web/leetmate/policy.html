LeetMate Privacy Policy

  • LeetMate is an extension that scrapes, tracks, and displays data from LeetCode on the LeetClub leaderboard.
  • The primary collection of data is the user's LeetCode username and their daily streak.
  • This data is stored in the browser's local storage and in the LeetClub servers when sent.
  • Data is only sent if the page is of the LeetCode website and the user has the extension enabled.
  • LeetMate does not collect any other data from the user.
  • The source code for LeetMate is available on GitHub - https://github.com/AttAditya/leetmate. (NOTE: The source code is the latest and testing versions of the extension and may not be the same as the version you are using.)
  • The source code for LeeClub is available on GitHub - https://github.com/AttAditya/leetclub
